# Electron Cloud Simulation

## Overview

Electron Cloud Simulation is a C++ project focused on visualizing a simplified model of electron motion and particle dynamics around an atomic center. The project is designed as an educational and experimental simulation that combines custom vector mathematics, particle entities.

It is useful as a study project for computer graphics, physical modeling, and object-oriented simulation design.

## Features

- Simulation of particle-like electron behavior
- Separate abstractions for electrons, protons, and intermediate structures
- Custom vector mathematics implementation
- OpenGL-based visualization
- Modular C++ project structure for further development

## Technologies

- C++
- Visual Studio solution/project structure
